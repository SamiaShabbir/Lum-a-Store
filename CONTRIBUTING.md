# Contributing to Lumea Store


Thanks for your interest! This is a demo project focused on clean structure and performance.


## Setup
1. Fork and clone
2. `npm i`
3. `npm run dev`


## Branching
- Features: `feat/<short-slug>`
- Fixes: `fix/<short-slug>`
